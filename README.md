# Deloitte Data Analytics Virtual Experience

## Project Overview
This project was completed as part of a virtual internship by Deloitte via Forage. The objective was to analyze telemetry and employee data to identify operational inefficiencies and equality issues.

## Task 1: Telemetry Data Analysis (Tableau)

### Problem Statement
Analyze machine telemetry data to identify downtime across factories and device types.

### Steps Performed
- Imported JSON telemetry dataset into Tableau
- Created a calculated field **"Unhealthy"** to estimate downtime
- Built visualizations:
  - Down Time per Factory (Bar Chart)
  - Down Time per Device Type (Bar Chart)
- Created an interactive dashboard
- Enabled filtering: selecting a factory dynamically updates device-level downtime

### Key Insight
- Identified the factory with the highest downtime
- Observed which device types contributed most to inefficiencies

---

## Task 2: Equality Analysis (Excel)

### Problem Statement
Evaluate employee equality across factories using equality scores.

### Steps Performed
- Analyzed Equality Score dataset
- Created a new column **"Equality Class"**:
  - Fair (±10)
  - Unfair (< -10 and > 10)
  - Highly Discriminative (< -20 and > 20)

### Key Insight
- Identified factories with potential discrimination issues
- Highlighted areas needing HR intervention
